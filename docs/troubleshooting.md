# Решение проблем

## Приложение не запускается

**Приложение уже запущено**
```
Application already running
```
Предыдущий процесс не завершился. Удалите lock-файл или дождитесь завершения.

**Нет интернета**
```
No internet connection
```
Проверьте подключение к сети.

**Qwen недоступен**
```
Qwen unavailable
```
Сервис chat.qwen.ai может быть недоступен. Проверьте вручную в браузере.

## Ошибки регистрации

**Форма не отправляется**
```
Failed to submit form
```
Возможные причины:
- Изменилась структура страницы Qwen (селекторы устарели)
- Капча/Cloudflare блокирует отправку
- Проблемы с сетью

**Письмо не пришло**
```
Email not received
```
- Увеличьте `mail.timeout` в `config.json`
- Проверьте CatchMail — возможна задержка доставки
- Проверьте, что email сгенерирован правильно

**Ошибка активации**
```
Activation failed
```
- Откройте ссылку активации вручную в браузере
- Возможно, ссылка уже истекла

**Logout не удался**
```
Logout failed
```
- Запустите повторно — logout выполнится автоматически
- Если проблема повторяется — очистите профиль браузера

## Диагностика

```bash
qf --debug
```

В режиме `--debug` логи выводятся в консоль. Используйте для поиска проблем.

Также доступна встроенная диагностика (пункт меню 5).
