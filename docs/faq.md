# FAQ

## Общие вопросы

**Можно ли использовать другой почтовый сервис вместо CatchMail?**
Да. Измените `mail.apiUrl` и `mail.domain` в `config.json`.

**Браузер открывается при каждом запуске?**
Нет. Браузер запускается один раз (lazy init) при первой операции.

**Сколько аккаунтов можно создать за раз?**
В пакетном режиме до 50. Ограничение можно изменить в исходном коде (`MAX_BATCH` в `src/services/batch.ts`).

**Как остановить процесс?**
- ESC в режимах ожидания (письмо, активация)
- Ctrl+C для полной остановки
- Пункт меню 0 (Выход)

**Можно ли использовать Qwen Forge на Windows?**
Через WSL. Проект не тестировался на нативной Windows.

## Технические вопросы

**Почему используется `domcontentloaded`, а не `networkidle`?**
Qwen поддерживает WebSocket-соединение постоянно. `networkidle` ждёт 0 активных соединений и никогда не наступит. `domcontentloaded` + `waitForSelector` — единственный надёжный паттерн.

**Что такое cloakbrowser?**
Библиотека для запуска Chromium с антидетект-настройками (stealth, fingerprinting, geolocation).

**Почему проект на Bun, а не на Node.js?**
Bun быстрее устанавливает зависимости, имеет встроенный TypeScript-рантайм и тест-раннер. Проект не имеет зависимостей от Node.js API.

**Как работает блокировка процесса?**
При запуске создаётся lock-файл в `/tmp/qwen-forge.lock`. Если файл уже существует — приложение завершается с ошибкой.
