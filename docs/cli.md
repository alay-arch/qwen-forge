# CLI

## Команды

| Команда | Описание |
|---------|----------|
| `qf` | Интерактивное меню |
| `qf --debug` | Запуск с TRACE-логами в консоль |
| `qf --help`, `-h` | Справка |
| `qf --version`, `-v` | Версия |

## Интерактивное меню

Запуск `qf` без аргументов открывает меню:

```
1  Создать аккаунт
2  Пакетное создание
3  Сессия
4  Статистика
5  Диагностика
6  Конфигурация
0  Выход
```

### 1 — Создать аккаунт

Создаёт один аккаунт Qwen:
- Генерирует одноразовый email через CatchMail
- Заполняет форму регистрации в браузере
- Подтверждает email по ссылке из письма
- Сохраняет логин и пароль в `data/accounts.json`

### 2 — Пакетное создание

Регистрация до 50 аккаунтов за один запуск. Введи количество — остальное автоматически.

Прогресс отображается в реальном времени. Можно отменить (Esc) в любой момент.

### 3 — Сессия

Показывает аккаунты, созданные в текущей сессии:
- Email и пароль
- Статус (успех / ошибка)
- Время создания

Можно очистить список сессии.

### 4 — Статистика

Общая статистика за всё время:
- Всего создано
- Успешных / ошибок
- Процент успеха
- Среднее время регистрации

### 5 — Диагностика

Проверяет систему:
- Интернет-соединение и DNS
- Наличие и запуск Chromium
- Недостающие shared libraries
- Доступность API CatchMail
- Диск и память
- Статус HTTP-сервера и браузера

### 6 — Конфигурация

Смена языка интерфейса (русский / английский).

## HTTP API

Сервер стартует автоматически при запуске `qf`.

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/ping` | Проверка работоспособности |
| POST | `/api/register` | Создать аккаунт |
| POST | `/api/logout` | Выйти из текущего аккаунта |

### Примеры

```bash
curl http://localhost:3030/api/ping
```

```bash
curl -X POST http://localhost:3030/api/register
```

```bash
curl -X POST http://localhost:3030/api/logout
```

Порт по умолчанию: `3030`. Настраивается в `config.json` → `server.port`.
