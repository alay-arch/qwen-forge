# CLI

## Команды

### `qf`

Запуск интерактивного меню.

```
1  Создать аккаунт
2  Пакетное создание
3  Сессия
4  Статистика
5  Диагностика
6  Конфигурация
0  Выход
```

### `qf --debug`

Запуск с расширенным логированием (уровень TRACE).

Вывод логов в консоль + запись в `logs/app.log`.

### `qf --help`

Показать справку.

### `qf --version`

Показать версию.

## Флаги

| Флаг | Описание |
|------|----------|
| `--debug` | Режим отладки (уровень TRACE, вывод в консоль) |
| `--help`, `-h` | Показать справку |
| `--version`, `-v` | Показать версию |

## HTTP API

Сервер запускается автоматически при старте приложения.

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/api/ping` | Проверка работоспособности |
| POST | `/api/register` | Регистрация аккаунта |
| POST | `/api/logout` | Выход и очистка сессии |

Пример:

```bash
curl -X POST http://localhost:9412/api/register \
  -H "Content-Type: application/json" \
  -d '{"email":"test@catchmail.io","password":"MyPass123!"}'
```

Порт по умолчанию: `9412`. Настраивается в `config.json`.
