# Changelog

## 0.1.0-beta

Первая публичная бета-версия.

### Добавлено
- Автоматическая регистрация аккаунтов Qwen
- Подтверждение email через CatchMail API
- Автоматическая активация по ссылке из письма
- Пакетное создание (до 50 аккаунтов)
- HTTP API для интеграции с внешними скриптами
- Диагностика системы
- Статистика (сессия + история)
- Два языка интерфейса (RU/EN)
- Graceful shutdown

### Архитектура
- `BrowserManager` — единственный владелец браузера
- `context.ts` — разрыв циклической зависимости AppContext
- logout в `finally` — гарантированная очистка после любой операции
- `waitUntil: 'domcontentloaded'` — предотвращение зависаний на WebSocket

### Установка
- `install.sh` — curl-to-bash инсталлятор
- `bin/qf` — глобальная команда
