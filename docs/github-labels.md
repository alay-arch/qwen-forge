# GitHub Issue Labels

Recommended labels for Qwen Forge issues and pull requests.

## Standard Labels

| Label | Color | Description |
|-------|-------|-------------|
| `bug` | `#d73a4a` | Something isn't working |
| `enhancement` | `#a2eeef` | New feature or request |
| `documentation` | `#0075ca` | Improvements or additions to documentation |
| `question` | `#d876e3` | Further information is requested |
| `good first issue` | `#7057ff` | Good for newcomers |
| `help wanted` | `#008672` | Extra attention is needed |
| `security` | `#e99695` | Security-related issue |
| `duplicate` | `#cfd3d7` | This issue or PR already exists |
| `invalid` | `#e4e669` | This doesn't seem right |
| `wontfix` | `#ffffff` | This will not be worked on |

## How to Apply

1. Go to your repository on GitHub
2. Open **Settings > Labels**
3. Create each label with the color code above
4. Delete unused default labels (e.g., `documentation`, if you prefer your own)
